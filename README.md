# Rubyspark

A ruby port of [Spark](http://zachholman.com/spark/)

## Installation

Add this line to your application's Gemfile:

    gem 'rubyspark'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubyspark

## Usage

    rubyspark 1 5 22 13 53
    => ▁▁▃▂▇
